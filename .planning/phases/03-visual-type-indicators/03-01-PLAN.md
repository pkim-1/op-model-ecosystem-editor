**JavaScript additions** (add before the `renderCard` function):

3. Add the TYPE_KEYWORDS constant and detectType function:

```javascript
// --- Type Detection ---
const TYPE_KEYWORDS = {
  analysis: ['analysis', 'data-science', 'jupyter', 'research', 'statistics', 'analytics'],
  tool: ['tool', 'cli', 'library', 'framework', 'utility', 'automation', 'dashboard'],
  visualization: ['visualization', 'dashboard', 'chart', 'graph', 'plot', 'dataviz'],
};

function detectType(topics) {
  if (!topics || topics.length === 0) return null;
  for (const topic of topics) {
    const lower = topic.toLowerCase();
    for (const [type, keywords] of Object.entries(TYPE_KEYWORDS)) {
      if (keywords.some(kw => lower.includes(kw))) {
        return type;
      }
    }
  }
  return null;
}
```

Keywords exactly per D-04. Type iteration order: analysis, tool, visualization (per D-04 order and RESEARCH.md Open Question 1 recommendation). First matching topic wins per D-05. Case-insensitive per D-07. Returns null for no match per D-06.
