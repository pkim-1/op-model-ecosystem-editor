- Compute `childOnlyTeams` = set of team IDs that appear in any `proc.children[*].teams` but NOT in `proc.teams`
- For each card in `state.cards`:
  - If `target.teams.includes(card.id)`: render normally with click handler (existing behavior)
  - Else if `childOnlyTeams.has(card.id)`: render with class `team-check-item inherited selected`, append " (sub)" text, NO click handler
  - Else: render normally with click handler (unselected, clickable to add to parent)

When `isSubPill`: keep existing behavior unchanged (only show that child's teams, all clickable).

The replacement code for the `state.cards.forEach(card => { ... })` block when `!isSubPill`:
```javascript
// Collect child-only teams for parent rollup display
const childOnlyTeams = new Set();
if (!isSubPill && proc.children) {
    proc.children.forEach(child => {
        (child.teams || []).forEach(t => {
            if (!target.teams.includes(t)) childOnlyTeams.add(t);
        });
    });
}

state.cards.forEach(card => {
    const item = document.createElement('div');
    item.className = 'team-check-item';
    const isDirectTeam = target.teams.includes(card.id);
    const isInherited = childOnlyTeams.has(card.id);

    if (isDirectTeam) {
        item.classList.add('selected');
    } else if (isInherited) {
        item.classList.add('selected', 'inherited');
    }

    item.textContent = card.name;
    if (isInherited) {
        const sub = document.createElement('span');
        sub.textContent = ' (sub)';
        sub.style.fontSize = '0.7rem';
        item.appendChild(sub);
    }

    if (!isInherited) {
        item.addEventListener('click', () => {
            const idx = target.teams.indexOf(card.id);
            if (idx >= 0) {
                target.teams.splice(idx, 1);
                item.classList.remove('selected');
            } else {
                target.teams.push(card.id);
                item.classList.add('selected');
            }
            deriveConnections();
            renderConnections();
        });
    }
    checklist.appendChild(item);
});
```
