## Implementation Details

### Task 1: Color-Matched Glow + Parent Dots
- `.sub-pill.active-pill` now uses `--active-color` and `--active-bg` CSS variables with fallback to accent-yellow

- Parent pills with children get small colored dots (6px circles) showing each child's assigned color
- `.process-pill.active-pill` also supports `--active-color` variable with fallback

### Task 2: RACI Matrix
- New `migrateTeams()` helper converts old format (array of card IDs) to new format (array of `{id, role}` objects)
- RACI badge appears next to each selected team in edit mode checklist

- Team name click toggles assignment with default role 'R'
- Present mode panel shows team chips with RACI role badges
- `deriveConnections()` extracts `.id` from team objects for connection logic
- `deleteCard()` updated to use `findIndex` for object-based team arrays
- `highlightTeamsForPill()` maps team objects to IDs before building highlight set
